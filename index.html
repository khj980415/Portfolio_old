SKILLS

인터넷 접근성, 웹 표준을 준수한 HTML을 작성하며 JavaScript를 통해 다채로운 인터랙션을 구현하고 있으며, 다양한 라이브러리를 활용하여 웹을 구현하고 있습니다.

  1. HTML & CSS & SASS

    HTML5 DTD 문법을 준수하고 웹 접근성을 높인 Semantic Page를 작성할 수 있습니다. 모바일 디바이스를 고려한 반응형 스타일을 구현하고 전처리 기능(SASS)를 이용해 보다 효율적인 CSS를 작성할 수 있습니다.

  2. JAVASCRIPT

    ScrollTrigger

    최신 ECMAScript 문법과 기능을 활용하여 JavaScript 코드를 보다 효율적이고 현대적으로 작성할 수 있으며, GSAP와 SwiperJS 등의 라이브러리를 사용하여 UX를 보다 인터랙티브하게 구현할 수 있습니다.

  3. REACT

    React를 활용한 사용자 인터페이스를 구현할 수 있으며 JSX, JavaScript 확장 XML을 작성할 수 있고 React와 Redus, Router, Axios 등의 라이브러리를 결합해 React를 더욱 효율적으로 활용할 수 있습니다.

  4. NODE.JS

    EJS

    Node.js를 사용하여 웹 서버, 데이터베이스 통신 등 다양한 서버 측 작업을 수행할 수 있습니다.

  5. MongoDB

    MongoDB를 통해 데이터베이스를 설계, 구축 및 관리할 수 있으며, 유연하고 확장 가능한 데이터베이스 솔루션을 구현할 수 있습니다.

  6. Git

    Git과 GitHub, aws를 사용하여 코드를 효율적으로 관리하고 협업, 코드 리뷰, 버전 관리 등의 다양한 작업을 수행할 수 있습니다.

  7. PROGRESSIVE WEB APP

    웹 기술을 사용하여 네이티브 앱과 유사한 사용자 경험을 제공하는 PWA를 제작할 수 있으며, Service Worker, 오프라인 캐싱, 푸시 알림 등의 기술을 활용하여 사용자에게 원활하고 향상된 UX를 제공할 수 있습니다.
